# Speed Maths App

A minimalist mental math trainer designed to improve calculation speed without using a calculator.

## Live Demo
https://speed-maths-app.vercel.app/

## Features
- Mental math tricks
- Timed 20-question speed quiz
- Difficulty levels (Easy / Medium / Hard)
- Multiple question types:
  - Multiplication
  - Squares
  - Percentages
  - Addition
- Score, streak, and best-score tracking

## Tech Stack
HTML • CSS • JavaScript

## Run Locally
```bash
git clone https://github.com/Namita-sha/Speed_maths_app.git
cd Speed_maths_app
open index.html

```
